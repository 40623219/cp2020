(軟體篇)Windows 7操作系統

總覽
Windows 7設計在Windows Vista中引入的許多技術上進行了改進，並改善了消費者的總體操作系統體驗。 Win7於2009年發布，是微軟對消費者明顯不喜歡Vista的一種回應，同時提供了可在多種平台上運行的強大，安全的消費者級操作系統，包括基於Apple Intel的計算機和Tablet PC。像Vista一樣，Win7設計的改進是使IT部門的操作系統部署和管理更加有效和安全。

Overview
Windows 7 design improved on many technologies introduced in Windows Vista and improved the overall OS experience for consumers. Released in 2009, Win7 was Microsoft’s response to consumers’ apparent dislike of Vista, while at the same time providing a robust, secure consumer grade OS that would run on multiple platforms including Apple Intel-based computer’s and Tablet PC’s. Like Vista, the improvements in Win7’s design were to make OS deployment and management more efficient and secure for IT departments.

總體而言，Win7成為了微軟希望Vista成為替代WinXP的工具。 Vista的一些主要功能包括改進的帶有索引的Windows搜索（位於“開始”菜單中的搜索框），用戶帳戶訪問（UAC），Aero功能和BitLocker。 Win7中包含的一些改進是清理後的任務欄，其中刪除了“快速啟動”，添加了“固定應用程序”以及將“顯示桌面”移至系統托盤的最右邊。默認情況下，稱為庫的特殊快捷方式旨在將來自多個來源的文檔，圖片和視頻組合在一起，只要它們可以被索引。包括Homegroup網絡基礎架構在內的網絡支持和安全性得到了改善，該網絡基礎架構允許對等LAN網段更輕鬆地共享文件，媒體和打印服務。 Win7還增加了對平板電腦，SSD技術和生物識別的專門支持。

Overall, Win7 became what Microsoft hoped Vista would be – as replacement for WinXP. Some of the main features continued from Vista include a refined Windows Search with Indexing (search box located in Start Menu), User Account Access (UAC), Aero features, and BitLocker. Some of the improvements included in Win7 are a cleaned up taskbar removing the Quick Launch, adding Pinned Applications, and moving Show Desktop to the far right of the System Tray. Specialized shortcuts called Libraries are designed by default to bring Documents, Pictures, and Videos together from multiple sources provided they can be indexed. Networking support and security is improved including the Homegroup networking infrastructure which allows a peer-to-peer LAN segment to more easily share files, media and print services. Win7 also added support specifically for tablets, SSD technology, and Biometric Authentication.

Windows Win7：版本和版本
Microsoft繼續使用與Vista中類似的命名結構，並發布了六個版本的Win7。但是，與XP和Vista不同的是，簡化版現已可供全球原始設備製造商（OEM）使用，而Home Basic則保留給新興經濟市場使用。四個零售版本分別是Home Basic，Home Premium，Professional和Ultimate。企業版仍命名為Enterprise。 Win7 Enterprise僅可通過使用批量許可證密鑰激活的Microsoft軟件保障購買。

Windows Win7: Versions and Editions
Microsoft continued with similar naming structure as in Vista and released six editions of Win7. However, unlike XP and Vista, the Starter Edition was now available to Original Equipment Manufacturers (OEMs) worldwide and Home Basic was reserved for emerging economic markets. The four Retail Editions are Home Basic, Home Premium, Professional, and Ultimate; with the Corporate Edition still named Enterprise. Win7 Enterprise is only available for purchase through Microsoft Software Assurance using Volume License Key activation.
